노트 활용 아이디어

어서 오세요! Evernote입니다 👋

Evernote는 노트 필기, 작업 관리, 프로젝트 진행, 자료 정리를 위한 최적의 장소입니다. 이제부터 소개해 드릴 인기 기능들을 활용해 정보를 캡처하고, 정리하고, 어디서나 액세스하세요.




아이디어 캡처

  • 멋진 아이디어를 모두 기록하세요
  • 좋아하는 글꼴로 노트 필기
  • 굵은 글씨, 기울임꼴, 밑줄, 하이라이트로 텍스트를 강조하거나, 텍스트에 색상을 추가하세요.


할 일 목록 만들기

첨부 파일 추가

문서, PDF, 사진, 동영상, 오디오 파일을 노트에 직접 끌어다 놓으세요. 서식 지정 모음에서 종이클립 아이콘을 클릭해 첨부파일을 추가할 수도 있습니다.

노트에 추가해 보세요:
  • 화이트보드와 손글씨 노트의 사진 — 이미지 안의 단어까지 검색이 가능합니다!
  • Evernote 모바일로 스캔한 영수증과 회계 서류
  • 동영상과 음성 노트
  • 프로젝트 자료와 PDF

오디오 녹음

서식 지정 모음에서 마이크 아이콘을 클릭해 음성 메모, 회의, 인터뷰, 강의를 Evernote에 직접 녹음하세요.



이미지와 PDF에 주석 달기

텍스트, 화살표, 하이라이트로 이미지와 PDF에 주석을 추가하여 아이디어를 전달하세요.
  • Mac, Windows, iOS, Android에서 사용할 수 있습니다. www.evernote.com/download로 이동하세요.
  • Evernote 베이직에서는 이미지에 주석을 추가할 수 있습니다. PDF 주석은 프리미엄 기능입니다.



웹 페이지를 Evernote에 직접 저장

Web Clipper 확장 프로그램을 사용해 인터넷에서 찾은 자료를 Evernote에 저장하세요.
www.evernote.com/webclipper에서 Web Clipper를 다운로드하세요.

노트북으로 노트 정리

노트북은 노트의 모음입니다. 새로운 노트는 다른 노트북을 지정하지 않는 한 기본 노트북에 저장됩니다. 추가 노트북을 만들어 주제별로 노트를 정리하세요. 다른 사람들의 노트 정리 방법을 참고하세요:

노트 공유

노트 오른쪽 상단 모서리의 공유 버튼을 클릭해 다른 Evernote 사용자들과 노트를 공유할 수 있습니다. 이메일이나 공유 링크를 통해서도 모든 사람과 노트를 공유할 수 있습니다.
